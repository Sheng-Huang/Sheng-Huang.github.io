第五章 控制流

  • Java 和 C/C++ 同属“自由格式”的编程语言，不像python有缩进等限制，但通常我们会在 Java 控制流程语句中采用首部缩进的规范，以便代码更具可读性。
  • Math 库的静态方法random()。该方法的作用是产生 0 和 1 之间 (包括 0，但不包括 1) 的一个 double 值。
  • do-while格式如下：
1
2
3

do 
	statement
while(Boolean-expression);
  • for所定义的变量的作用域仅在for循环体内。
  • 在 Java 中逗号运算符（这里并非指我们平常用于分隔定义和方法参数的逗号分隔符）仅有一种用法：在 for 循环的初始化和步进控制中定义多个变量。注意：在 Java 中，仅允许 for 循环在控制表达式中定义变量。
  • for-in(Python 已有类似的先例，如：for x in sequence），将集合或数组的每一个元素赋值给变量：
1

for(float x : f) {}
  • 在 Java 中有几个关键字代表无条件分支，这意味无需任何测试即可发生。这些关键字包括 return，break，continue 和跳转到带标签语句的方法，类似于其他语言中的 goto。
  • 如果在方法签名中定义了返回值类型为 void，那么在代码执行结束时会有一个隐式的 return。 也就是说我们不用在总是在方法中显式地包含 return 语句。 注意：如果你的方法声明的返回值类型为非 void 类型，那么则必须确保每个代码路径都返回一个值。
